# Encrypted Modules Scripts

Cryptographic utilities and encrypted module handling scripts.

## 📁 Scripts

- **crypt.ts** - Cryptographic utilities for encrypted modules

## 🚀 Usage

### Cryptographic Operations
```bash
# Run with Node.js
npx ts-node scripts/encrypted-modules/crypt.ts
```

## 🔐 Features

- Encryption/decryption utilities
- Key management
- Module encryption handling
- Secure data processing

## 📝 Notes

- Use with encrypted modules in `src/encrypted-modules/`
- Ensure proper key management and security practices
- Works with the git encrypted modules workflow
