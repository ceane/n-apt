# Processing Scripts

Data processing utilities and CI/CD helper scripts.

## 📁 Scripts

- **check_changes.sh** - Check for file changes before commits
- **ci_timing.sh** - CI/CD timing and performance monitoring
- **indent_output.sh** - Code formatting and indentation
- **kill_blockers.sh** - Kill blocking processes
- **test_npm_width.sh** - Test npm package width constraints
- **test_term.sh** - Terminal compatibility tests

## 🚀 Usage

### Check Changes
```bash
./check_changes.sh
```

### CI Timing
```bash
./ci_timing.sh
```

### Code Formatting
```bash
./indent_output.sh file.js
```

### Kill Blockers
```bash
./kill_blockers.sh
```

## 📝 Notes

- All scripts are executable
- Use relative paths from project root
- Check individual script help for detailed options
